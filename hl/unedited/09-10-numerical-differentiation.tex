\subsection{Numerical differentiation}
HL p 546
\obeylines
Involves the derivatives of function f with respect to x.

1st order means $\frac{d}{dx}(y)=y'(x)=f'(x)$

Differential equation like
$f(x,y)\cdot \frac{d}{dx}(y)+g(x,y)=0$, where $y=y(x)$.
solve it for y.
$\frac{d}{dx}(y)=\frac{-g(x,y)}{f(x,y)}$ is slope of tangent
$y=\int \frac{-g(x,y)}{f(x,y)}dx$
general solution with constant of integration c.
The set of tangents at all points (x,y)
is called the slope field of differential equation.

An isocline is set of all points on slope field at which the tangents have the same gradient.

Euler's method:
start at point ($x_0$,$y_0$)
with step size $h=0.1$ or $0.001$ ... or 0.5
$x_{n+1}=x_n+h$ and $y_{n+1}=y_n+f(x_n,y_n)$
where $f(x_n,y_n)=\frac{d}{dx}(y)$
gives approximate solution to initial value problem. (IVP)

Example. Find value of y at $x=5$ given that 
$\frac{d}{dx}(y)=\sqrt{x\cdot y+1}$ and curve passes point $(4.5;1)$.
