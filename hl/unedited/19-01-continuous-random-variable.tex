X is measured like decminal numbers
1) probability density (like frequency) function $f(x)\geq 0$ and $\int_{-\infty}^{\infty}f(x)=dx=100\%=1$
2) mean$=\mu=E(X)=$expected value of random variable
$\int_{a}^{b}(x\cdot f(x))dx$
3) variance $\sigma^2=\int_{a}^{b}((x-\mu)^2\cdot f(x))dx$
4) standard deviation $\sigma=\sqrt{\sigma^2}=\sqrt{variance}$
5) variance = $E(x^2)-(E(x))^2$
6) cumulative distribution function
$F(X)=\int_{-\infty}^{x}f(x)dx=P(X<x)$
and $F'(x)=f(x)$
7) Median m can be solved from
$\int_{-\infty}^{m}f(x)dx=50\%=\frac{1}{2}$
8) mode=most typical value x, with maximum $f(x)$
